The tasks are solved by test driven development. This project is for exercise purpose only.

## Build with

- Intellij v2022.2.1
- Apache Maven v4.0.0
- OpenJDK v18.0.1.1
- JUnit v5

## Usage

Run unit tests:

```sh
mvn test
```
