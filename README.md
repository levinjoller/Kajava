The tasks are solved by test driven development. This project is for exercise purpose only.

## Build with

- Intellij v2023.1
- Gradle v8.2.1
- OpenJDK v18.0.1.1
- JUnit v5.9.3

## Usage

Run unit tests:

```sh
./gradlew build
```
